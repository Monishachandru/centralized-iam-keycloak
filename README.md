# Centralized IAM with Keycloak

## Overview

This repository provides a centralized **Identity and Access Management (IAM)** solution built with **Keycloak**, enabling secure and consistent authentication and authorization across multiple applications and services.

The solution supports **Single Sign-On (SSO)**, **Role-Based Access Control (RBAC)**, and **Multi-Factor Authentication (MFA)** using industry-standard protocols such as **OAuth 2.0** and **OpenID Connect (OIDC)**.

---

## Key Features

* Centralized authentication and authorization
* Single Sign-On (SSO) for multiple applications
* Role-Based Access Control (RBAC)
* Multi-Factor Authentication (MFA) using OTP
* OAuth 2.0 and OpenID Connect compliance
* JWT-based token authentication
* Realm-based isolation and multi-tenancy support
* Easy integration with web applications, APIs, and microservices

---

Keycloak acts as the central identity provider responsible for authentication, authorization, token issuance, and user management.
